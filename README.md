# 匿名评报APP - 基础版本设计

## 核心理念
- **完全匿名**：无需注册，无身份追踪
- **真实评价**：多维度专业评分体系
- **记者互助**：提升行业整体水平

## 技术栈
- 前端：HTML + Tailwind CSS + Vanilla JS
- 后端：LocalStorage（基础版）/ 后续可接后端
- 设计：深色主题，专业、严肃、可信

## 页面结构
1. 首页 - 文章列表 + 评分入口
2. 提交文章 - 录入要评价的新闻
3. 评分页面 - 多维度打分 + 评论
4. 文章详情 - 查看评分统计

## 评分维度（新闻专业标准）
1. 事实准确性 (Accuracy)
2. 信息完整性 (Completeness)
3. 客观公正性 (Objectivity)
4. 时效性 (Timeliness)
5. 写作质量 (Writing)
6. 社会价值 (Social Value)

## 匿名机制
- 使用设备指纹生成匿名ID（非追踪用途）
- 无手机号、无邮箱、无第三方登录
- 数据本地存储，保护用户隐私
